# Advent of Code 2020

**Description**

    input files are available at `inputs/$day_num.txt`
    code is in `libs/D$day_num.ex`

    to run in iex

```sh
    > iex -S mix
    D1.run
    D2.run

    open D1.run    # to go directly to code from iex
```
